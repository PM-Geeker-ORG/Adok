# 《50》心理认知偏差
认知偏差这个词语早就有所听闻，但是初次了解《50个认知偏差》则是从马斯克的推荐分享中开始，我因此好奇并去学习，最后整理了别人对50个认知偏差的总结。这些认知偏差都是一些我们日常的心理活动，以致于我们难以察觉并养成习惯，只要我们了解到我们存在着这些心理的认知偏差，并能在日后的认知过程中加以调整，不再让偏差成为我们的胖脚石。
